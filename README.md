# Pong Game

A simple browser-based pong-style tennis game built with HTML, CSS, and JavaScript.

## How to play

- Open `index.html` in your browser
- Move your mouse to control the left paddle
- Score points by getting the ball past the computer paddle
- Click the canvas to restart after a win

## Features

- Canvas-based game rendering
- Basic AI opponent
- Score tracking
- Sound effects for paddle hits and win/lose events
- Smooth paddle movement and ball physics

## Tech

- HTML
- CSS
- JavaScript
